# 🚀 NASA Explorer

Projeto desenvolvido com FastAPI (Python) e HTML, CSS e JavaScript para consultar informações públicas da NASA.

## Funcionalidades

* Imagem Astronômica do Dia (APOD)
* Imagens de Marte
* Asteroides Próximos da Terra
* Catálogo de Asteroides
* Imagens da Terra (EPIC)
* Biblioteca de Imagens da NASA
* Clima Espacial
* Eventos Naturais

## Tecnologias Utilizadas

### Back-end

* Python
* FastAPI
* HTTPX

### Front-end

* HTML
* CSS
* JavaScript

## Como Executar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Inicie a API:

```bash
uvicorn main:app --reload
```

Acesse:

```text
http://127.0.0.1:8000
```

Documentação da API:

```text
http://127.0.0.1:8000/docs
```
